CitiBike with Tableau!

In this project, I downloaded CitiBike Trip History Data for January and February 2018. I cleaned up and combined the data sets using Pandas and saved the data into a csv file. Using Tableau, I built different dashboards that shows popularity of stations, analysis of peak hours and bike usage by demographics and user types.

My Experience

Aside from small exercises, this is the first project on Tableau that I've done. I love Tableau for how easy it is to use but with the convenience, I feel like I had to give up some control. I played around with calculated fields, aliases, filters and actions. There were a lot of points where I got frustrated but I am satisfied with the end result. One thing I noticed with Tableau is that because it is so easy to build charts just by dragging and dropping the pills, I sometimes had to step back and check if the chart is showing what I want it to show.
